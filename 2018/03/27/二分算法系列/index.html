二分法

通用二分写法：

LeetCode 69 Sqrt(n)
Input: 4
Output: 2
Input: 8
Output: 2
Explanation: The square root of 8 is 2.82842…, and since we want to return an integer, the decimal part will be truncated.

public int Sqrt(int n){
    int l=0;h=n;
    while(l<=h){
        int mid=l+(h-l)/2;
        if(mid^2==n)return mid;
        if(mid^2>n)h=mid-1;
        else l=mid+1;
    }
    return h;
}
