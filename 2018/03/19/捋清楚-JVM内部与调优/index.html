JVM是什么？
Java Virtual Machine 区别于JMM（Java Memory Model）
Sun公司开发的，用于屏蔽语言与平台之间的隔阂，实现了Java语言的跨平台性。
此跨平台性并不是针对Java语言，而是适配Java程序编译后的字节码文件（.class文件）注：JVM平台之上还有其他语言（Scala等）
